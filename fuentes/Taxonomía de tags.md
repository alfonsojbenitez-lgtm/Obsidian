# Taxonomía de tags del vault

Referencia completa del sistema de etiquetas. Usar siempre estos tags para mantener consistencia.

---

## Por tipo de nota

| Tag | Uso |
| --- | --- |
| `#diario` | Notas diarias |
| `#semanal` | Notas semanales |
| `#mensual` | Notas mensuales |
| `#reunión` | Notas de reunión |
| `#email` | Correos capturados |
| `#sistema` | Notas de configuración del vault |

## Por proyecto

| Tag | Uso |
| --- | --- |
| `#GALIAT` | Todo lo relacionado con el ensayo GALIAT |
| `#ensayo-clínico` | Ensayos clínicos en general |
| `#huella-hídrica` | Subproyecto huella hídrica |
| `#dieta-atlántica` | Dieta atlántica |
| `#SERGAS` | Trabajo en el Servizo Galego de Saúde / CHUS |
| `#análisis-clínicos` | Especialidad y actividad de laboratorio |
| `#IA` | Inteligencia artificial (ambas líneas) |
| `#medicina` | IA aplicada a medicina y clínica |
| `#herramientas` | Configuración de herramientas IA personales |

## Por tipo de contenido

| Tag | Uso |
| --- | --- |
| `#fuente` | Referencias y recursos externos |
| `#artículo` | Artículos científicos |
| `#protocolo` | Protocolos de estudio |
| `#bibliografía` | Índices bibliográficos |
| `#búsqueda` | Investigaciones activas |

## Por disciplina

| Tag | Uso |
| --- | --- |
| `#estadística` | Análisis y métodos estadísticos |
| `#R` | Scripts y recursos de R Software |
| `#data-science` | Ciencia de datos |
| `#investigación` | Investigación científica general |
| `#nutrición` | Nutrición y dietética |

## Por estado

| Tag | Uso |
| --- | --- |
| `#pendiente` | Requiere acción urgente |
| `#en-curso` | Trabajo en progreso |
| `#completado` | Finalizado |
| `#archivado` | Ya no activo |

## Por tipo de persona

| Tag | Uso |
| --- | --- |
| `#persona` | Contacto general |
| `#investigación` | Miembro de grupo de investigación |

---

> [!tip] Regla general
> Menos es más. Usa 2-4 tags por nota. Si dudas entre dos tags, usa el más específico.
