# Inbox

Zona de descarga rápida. Todo lo que llega aquí debe procesarse y moverse a su carpeta definitiva.

> [!tip] Regla de oro
> El inbox no es un destino, es un tránsito. Procésalo al menos una vez por semana.

## Protocolo de procesamiento

1. ¿Es accionable? → muévelo a [[proyectos/]]
2. ¿Es una referencia? → muévelo a [[fuentes/]] o [[búsquedas/]]
3. ¿Es sobre una persona? → muévelo a [[personal/]]
4. ¿No sirve? → elimínalo
