# IA — Inteligencia Artificial

## Contexto

Proyecto transversal que agrupa dos líneas de trabajo con IA:

1. **IA aplicada a medicina y estadística** — investigación, métodos y herramientas de IA para análisis clínicos, diagnóstico asistido, procesamiento de datos biomédicos.
2. **IA personal** — configuración y optimización de herramientas IA adaptadas al flujo de trabajo propio (Claude Code, LLMs, automatizaciones del vault).

## Líneas de trabajo

### IA en medicina y estadística

| Área | Descripción |
| ---- | ----------- |
| **Diagnóstico asistido** | Modelos predictivos en análisis clínicos |
| **NLP biomédico** | Procesamiento de texto clínico, informes |
| **ML en ensayos clínicos** | Métodos de ML aplicados a datos de ensayos |
| **Estadística bayesiana** | Alternativas bayesianas a métodos frecuentistas |

### IA personal y herramientas

| Herramienta | Uso |
| ----------- | --- |
| **Claude Code** | Segundo cerebro, vault Obsidian, análisis estadístico |
| **R + tidymodels** | ML y estadística avanzada |
| **Quarto** | Documentos reproducibles con IA |
