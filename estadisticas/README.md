# Estadísticas

Análisis estadísticos, modelos, resultados y reportes. Cada nota documenta un análisis concreto vinculado a su proyecto o fuente de datos.

> [!tip] Convención de nombres
> Usa el formato `YYYY-MM-DD - Nombre del análisis.md` para facilitar el orden cronológico.

## Relación con el resto del vault

- Los análisis nacen de [[proyectos/README|proyectos]] activos → enlaza siempre al proyecto padre
- Los métodos y técnicas nuevas van a [[búsquedas/README|búsquedas]] para investigarlos
- Las referencias estadísticas y cheatsheets van a [[fuentes/README|fuentes]]
- Los scripts `.R` y documentos `.qmd` se referencian desde aquí con wikilinks
- Usa la plantilla [[plantillas/Análisis estadístico]] para cada nuevo análisis
